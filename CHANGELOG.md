# 1.0.2 - 13/04/2026

## Features

- Added support for YAML input format
- Added ![] to the output of encrypted strings for better readability
- Added handling of ![] in the input string to allow decryption of previously encrypted values

## Improvements

- Shrinked the application window to 600x600 for a more compact interface

# 1.0.1 - 04/04/2026

Initial release of the Java Secure Properties App

## Features

- Supports encryption and decryption of strings using all algorithms and modes available in the Java Secure Properties library
- Provides a user-friendly interface for inputting data and viewing results
- Supports random IV generation for enhanced security during encryption
- Embarks a Java Secure Properties library JAR file for executing the secure properties operations
- Allows users to specify the path to an external JAR file for executing the secure properties operations
